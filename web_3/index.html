La programación informatica es el proceso de diseñar, depurar y mantener el código fuente de programs computacionales. El código fuente es escrito en lenguaje de programación

El proposito de la programción es crear programas que exhiban un comportamiento deseado. El proceso de escribir código requiere frecuentemente conocimientos en carias areas distintas, a demás del dominio del lenguaje a utilizar, algoritmos especificos especializados y lógica formal
